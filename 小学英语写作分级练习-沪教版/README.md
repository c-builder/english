> **适用教材**：上海教育出版社（沪教版 / 上教版）牛津英语 · 义务教育教科书（教育部审定 2013）
> **覆盖范围**：小学 1—6 年级全部写作能力训练
> **设计依据**：分阶段训练四步法 —— 写对单词 → 写对句子 → 写连句子 → 写小段落
> **篇幅**：每个年级 ≥ 6 页（A4），含练习、参考答案、范文、高频词块、批改要点

### 四阶段训练法（贯穿全部年级）

| 阶段 | 训练目标 | 输出长度 |
|------|----------|----------|
| 第 1 阶段 · 写对单词 | 拼写、大小写、单复数 | 单词 |
| 第 2 阶段 · 写对句子 | 主谓完整、标点、a/an | 1 句 |
| 第 3 阶段 · 写连句子 | 连接词、句间顺序 | 2—3 句 |
| 第 4 阶段 · 写小段落 | 结构完整、围绕主题 | 3—6 句以上 |

### 批改三原则

1. 先表扬，后纠错；保护写作兴趣优先。
2. 一次只抓 1—2 类错误，别满篇红。
3. 重点抓高频错误：大小写、句号、is/are、a/an、单复数、主谓一致。

### 写作"五查"口诀

> 一查大写 → 二查句号问号 → 三查主谓 → 四查单复数 → 五查意思连贯
